# Visual editor image coverage

The homepage visual editor provides explicit controls for every non-product photograph currently rendered on the homepage.

## Editable foreground images

- Story image
- Give One purpose image

Each foreground image supports:

- Global Media Library selection
- Governed image upload
- Accessible image description
- Focal point: top, center, bottom, left, or right
- Fit: cover or contain
- Overlay: none, light, medium, or strong

## Editable background images

- Hero background
- Book background
- Churches and Ministries background

Backgrounds continue to use the existing Change Background controls and safe section presets.

## Governed elsewhere

Product-card images remain managed through Catalog and Media because they must stay associated with product records, variants, pricing, and product-accuracy review. Operational forms and icons remain structurally governed.
